ClassAve Loop Extension


Thank you for downloading the ClassAve Loop Extension.


Features

The Class Ave Loop Extension is a Google Chrome Extension that provides an additional features which allow incoming students to see past class grade averages for a particular class taught by a particular teacher.

Terms and Conditions

  • By downloading this Chrome extension, you are allowing SchoolLoop to access ALL your grades ANONYMOUSLY.
  • This extension allows you to check out your class' average. Other users of this extension WILL see your grades, but they will not see that those test, homework, or any assignment scores belong to you. You will remain completely anonymous.
  • You will only be able to access and use this extension if you have a valid School Loop account.
  • You have to enable your own anonymity. If you don't enable that, the extension won't work.
  • This extension is not made for hacking into fellow students' accounts. You can't access classmates' records without also accessing their password, which CANNOT be accessed through this Chrome extension.

Purpose

The purpose of this extension is to see the class average for a particular assignment, test, homework, etc. When you are enabled to use this extension, you can see other users' grades. However, you won't know who those other users are, and vice versa. Other users can see the grade you got on an assignment, test, homework, etc. but they won't know who you are. The only thing you know essentially about the person who got that grade is that they are in the same class period as you are and have the same teacher.
